# Ghizer Union Welfare Organization — Member Portal

A self-contained web app for managing cooperative savings, loans, and member accounts.

## Features
- Landing page with organization branding
- Manager login (full dashboard: members, savings, loans, interest, accounting, reports, audit log, settings)
- Member login via CNIC + password (set by the manager)
- Member portal: profile, balances, loan dues, and downloadable statements (PNG/PDF)
- Responsive design for mobile and desktop

## Usage
Open `index.html` in any modern browser, or enable **GitHub Pages** on this repo (Settings → Pages → deploy from the `main` branch) to host it online.

First-time setup: click **Manager Login** on the landing page to create the manager account, then add members from the Members tab.
